# 🏠 Airbnb Tableau Dashboard Project

## 📊 Project Overview

This project is an interactive Tableau dashboard analyzing Airbnb listing data.  
The dashboard provides insights into revenue trends, pricing patterns, bedroom distribution, and geographic price variations.

The goal of this project was to practice data visualization, storytelling, and business insight generation using Tableau.

---

## 📌 Dashboard Insights

### 1️⃣ Revenue for Year
- Visualizes total revenue trends across weeks in 2016
- Shows seasonal fluctuations and growth patterns
- Helps identify high-performing periods

### 2️⃣ Distinct Count of Bedrooms
- Displays the distribution of listings by number of bedrooms
- Majority of listings are 1-bedroom properties
- Useful for understanding supply distribution

### 3️⃣ Average Price Per Bedroom
- Compares average price across bedroom categories
- Shows how price scales with property size
- Highlights market positioning for larger listings

### 4️⃣ Average Price by Zip Code
- Compares pricing variations across different zip codes
- Identifies premium and lower-cost areas
- Useful for investment and pricing strategy analysis
